Code-Club Etiquette:

  1. Make sure that your query is in the right channel.
  1. A query regarding Java should be in the Java channel, queries about python and data modelling in the data modelling channel and so on. If you are not sure where you should post your query, direct message admin with your query.
  1. In your profile, add your OS details and  your programming language in the status field. This will help us respond to your queries faster.
  1. Refer to admin’s profile in case you are confused about this.
  1. Do not tag more than 2 people in your query. 
  2. Please make sure that you do not repeatedly post the same message in multiple channels.
  1. If you have direct-messaged someone, you will get a response.
  1. Please don’t publicly criticize anyone. It would be great if you could help out others with their queries if you know the solution to a particular issue.
  2. Formatting manners:
  1. Please don’t use caps lock while typing your queries (IT IS CONSIDERED SHOUTING ON THE INTERNET).
  2. Please don’t use multiple question marks or an excess of punctuations!!!!!!!!!!!???

  1. You are free to do random chitchat in the random channel.
  2. General channel is for the one-way communication with the Code-Club team. Don’t message anything on this channel.
